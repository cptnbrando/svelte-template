# A Svelte Project Template.

Uses
- Vite
- Svelte
- TailwindCSS
- SCSS
- JavaScript (fuck TypeScript lol)
- i18n (svelte-i18n)
- Icons (@lucide/svelte)
- Testing
  - Vitest
  - JSDOM
  - @testing-library/svelte
  - @testing-library/jest-dom

# Getting Started

git clone this repo
```
git clone https://github.com/cptnbrando/svelte-template.git
```

install
```
cd svelte-template
npm i
```
